# Financial Manager — Documentação

Bem-vindo(a) à documentação do **Financial Manager**. Este repositório prioriza regras explícitas, decisões registradas e leitura clara em qualquer leitor de Markdown.

---

## Como usar esta documentação
1. **Entenda o produto**: comece em [wiki/ideias.md](wiki/ideias.md).
2. **Regras de negócio**: leia [wiki/regras-de-negocio.md](wiki/regras-de-negocio.md) antes de implementar qualquer comportamento.
3. **Modelos de dados**: consulte [wiki/modelos-json.md](wiki/modelos-json.md) para persistência.
4. **Pendências**: toda dúvida não definida deve ir para [pendencias.md](pendencias.md).
