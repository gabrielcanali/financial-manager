## Contexto
O salário é uma entidade própria com pagamento final e possível adiantamento. A API deve persistir `salary.json` e gerar projeções conforme o mês, respeitando a pendência sobre confirmação automática.

---

## Escopo da Tarefa
### Inclui
- Criar e atualizar `salary.json` conforme o modelo.
- Gerar projeções de adiantamento e pagamento final para o mês.
- Calcular adiantamento como percentual do salário quando habilitado.

### Não inclui (importante)
- Confirmação automática das transações de salário (pendência P-001).
- Tipos de adiantamento além de percentual.
